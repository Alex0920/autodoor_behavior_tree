**新增内容**

- **OCR智能优化系统**：全面升级OCR识别能力，大幅提升识别准确率和定位精度
  - 新增"自动调优"预处理模式：系统自动分析图像特征并选择最佳预处理参数
  - 新增"自适应"预处理模式：使用CLAHE对比度增强和自适应二值化，适用于低对比度、渐变背景场景
- **窗口绑定功能**：支持将行为树绑定到特定窗口，实现后台监控
  - 在开始节点属性面板中启用"绑定窗口"选项
  - 从下拉列表选择目标窗口
  - 后台截图：使用 PrintWindow API 截取后台窗口图像
  - 自动坐标转换：进行识别和点击时，会将屏幕绝对坐标与窗口相对坐标自动转换
  - 智能窗口切换：执行动作前自动切换到绑定窗口，执行后恢复原窗口
- **文本提取节点**：新增文本提取条件节点，通过 OCR 识别屏幕文字并提取到黑板变量
  - 支持全文提取和关键词提取两种模式
  - 支持中英文、繁体中文 OCR 识别
  - 支持图像预处理模式（默认/复杂色彩）
  - 提取的文本自动保存到黑板变量，供文本输入节点使用
  - 可选保存全部 OCR 识别文本和检测位置
- **文本输入节点**：新增文本输入动作节点，支持三种输入模式
  - 文本提取值模式：从黑板变量获取文本（与文本提取节点配合使用）
  - 预设文本模式：从预设文本列表中获取，支持顺序/随机执行
  - 文件模式：从文本文件读取内容
  - 支持输入前清空、输入间隔设置、保存输入文本等功能
- **条件节点预览检测**：在属性面板配置参数底部添加"预览检测"按钮，无需运行行为树即可验证检测条件
  - 支持 OCR 检测、图像匹配、颜色检测、数字比较、变量判断节点
  - 结果直接输出到运行日志面板

**优化内容**

-  **优化了组合节点失败重试的重复间隔**：现在失败重试的执行间隔也可通过重复间隔控制
- **优化画布粘贴功能**：现在粘贴节点时会出现在画布目前焦点中心，且不会与其他节点

**修复内容**

- **修复小字体放大失真问题**：协调预处理放大与RapidOCR放大，避免双重放大导致失真
- **修复脚本节点在并行节点下几率无法结束运行的问题**
- **修复当文字检测过快时，会被误判成功的问题**
- **修复win键无法正确录制和自动按键的问题**
- **修复在修改按键节点的按键后，无法使用画布快捷键的问题**
- **修复按键节点无法绑定Delete、BackSpace等画布快捷键的问题**
-  **修复变量判断无法正确使用的问题**

*  **数字比较节点功能修复**：修复了多个数字比较节点功能上的BUG，包括参数无效、正则匹配无效、无法匹配负值等问题
